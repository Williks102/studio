# **App Name**: Le Lof Website

## Core Features:

- Homepage Carousel: Display a rotating carousel of dish images with the tagline: 'Le Lof: African vibes'.
- Social Media Follow Buttons: Implement 'Follow' buttons linking to Le Lof's LinkedIn, Instagram, and Facebook accounts to increase visibility and followers.
- "About Us" Section: Show the story of Le Lof. Originating from a passion for African flavors and a desire to enhance them, Le Lof is where tradition meets modernity. Each dish tells the story of a continent rich in colors, spices, and emotions.
- Menu Display: Display the breakfast, lunch, dinner, and drink menus. Use images of main dishes for each category.
- Formula display: Show brunch, private culinary experiences and buffet formula.
- Booking Request Form: Implement a booking request form.
- Quote Request Form: Implement a quote request form (including name, contact info, email, chosen formula, number of people).

## Style Guidelines:

- Primary color: Saturated brown (#A66321), referencing the colors and earthiness of the African continent and cuisine.
- Background color: Desaturated light tan (#F2E7D5), maintaining a natural feel while creating separation in the layout.
- Accent color: Golden yellow (#D4A63A), for highlighting key elements and creating visual interest, while retaining the overall theme.
- Font pairing: 'Playfair' (serif) for headings to evoke sophistication and 'PT Sans' (sans-serif) for body text.
- Use minimalist icons that reflect African culture.
- The layout must evoke https://parentheseabidjan.com/
- Subtle transitions for image carousel and menu item hovers.
